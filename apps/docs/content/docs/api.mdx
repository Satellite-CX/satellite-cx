# API Reference

The Satellite CX API is built with **tRPC** for type-safe communication between client and server.

## Authentication

API endpoints use session-based authentication with Better-auth.

## Endpoints

### Organizations

- Get organization details
- Manage organization members
- Organization settings

### Tickets

- Create new tickets
- Update ticket status
- Assign tickets to team members
- Add comments and attachments

### Customers

- Customer profiles
- Customer history
- Customer communication

## Base URL

```
http://localhost:8000/trpc
```

All API calls are made through the tRPC client for type safety.
