# Getting Started

Welcome to **Satellite CX**, an open source customer support ticketing system built as a modern, self-hosted alternative to platforms like Zendesk.

## What is Satellite CX?

Satellite CX is a customer support platform that provides:

- **Multi-tenant ticketing system** for organizations
- **Modern tech stack** for developers
- **Full data ownership** for your business
- **Extensive customization** capabilities

## Quick Start

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Start the development server with `pnpm dev`
4. Open your browser to `http://localhost:3000`

## Features

- Ticket management
- Customer profiles
- Team collaboration
- Tags and priorities
- Audit trails
- Multi-tenant architecture
