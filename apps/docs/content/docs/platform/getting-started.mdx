# Getting Started

Welcome to Satellite CX! This guide will help you get up and running with your customer support platform.

## Quick Start

1. **Installation** - Set up Satellite CX on your infrastructure
2. **Configuration** - Configure your organization settings
3. **First Ticket** - Create and manage your first support ticket
4. **Team Setup** - Add team members and set permissions

## What You'll Learn

- How to install and configure Satellite CX
- Basic ticket management workflows
- Setting up your team and permissions
- Integrating with your existing tools

## Prerequisites

- Node.js 18+ or Docker
- PostgreSQL database
- Basic understanding of customer support workflows
